# Ubuntu Wear KE

Modern React + TypeScript storefront and marketplace concept for buying and selling new and pre-loved attires in Kenya.

## Pages

- Homepage
- Shop / Marketplace
- Product Detail
- Sell Your Clothes
- Seller Dashboard
- User Account
- Cart / Checkout
- About Us
- How It Works
- Blog / Fashion Tips
- Contact Us
- FAQ
- Privacy Policy
- Terms & Conditions

## Scripts

- `npm install`
- `npm run dev`
- `npm run build`
